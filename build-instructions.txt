Build instructions:

1. sudo port install qt6
2. sudo port install qt6-qttools
3. sudo port install wxWidgets-3.0
4. sudo port select --set wxWidgets wxWidgets-3.0
5. tar xvf aethersx2-lgpl-linux-mac.tar.gz
6. cd aethersx2-lgpl
7. mkdir build
8. cd build
9. cmake -DCMAKE_BUILD_TYPE=Release -DQT_BUILD=ON -DUSE_OPENGL=OFF -DQt6_DIR=/opt/local/libexec/qt6/lib/cmake/Qt6  ..
10. make

Application is located in pcsx2-qt/AetherSX2.app.
